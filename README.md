# SCT_SD_2 – Random Number Guessing Game

## Description
This is a Java-based GUI application that generates a random number between 1 and 100.
The user is asked to guess the number, and the application provides feedback such as
"Too High", "Too Low", or "Correct".

The application is completely user-driven and provides a simple and user-friendly
graphical interface using JavaFX.

## Features
- Generates a random number automatically
- Accepts user input through GUI
- Provides instant feedback for each guess
- Reset option to start a new game
- Easy-to-use interface

## Technologies Used
- Java (JDK 21)
- JavaFX
- IntelliJ IDEA

## How to Run the Project
1. Open the project in IntelliJ IDEA
2. Ensure JavaFX SDK is installed
3. Set VM options as:
   --module-path "C:\Program Files\Java\javafx-sdk-21.0.10\lib" --add-modules javafx.controls
4. Run the MainApp.java file
